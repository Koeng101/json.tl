A lightweight JSON library for Teal. Inspired by / translated from [rxi's lua library](https://github.com/rxi/json.lua)

### Building and testing
To build from source, run:
```
cyan build
```

To test from source, run:
```
busted -c -m "./build/?.lua"
```
